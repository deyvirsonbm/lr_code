

# Desafio

#### Dependencies
* Ruby
* Bundle

#### Setup
* bundle install

#### Run
* Na raiz do projeto executar cucumber

#### Report
* Para criar um report automatico do cucumber executar o comando "cucumber desafio.feature --publish"
